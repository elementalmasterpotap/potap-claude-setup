Вайбкодинг-запрос: $ARGUMENTS

## Принцип:

```
желаемый результат (человеческий язык)
    │
    ▼
найти нужные константы / параметры / код
    │
    ▼
изменить только их
    │
    ▼
задеплоить
    │
    ▼
ждать фидбека
```

## Правила:
- Менять ТОЛЬКО прямо связанное с запросом — минимальный импакт
- Не трогать критические зависимости (детекторы, топология, балансные значения)
- Сохранить prev-версию до изменения
- Деплоить без интерактива: `-NoPrompt` / `--yes`

## Хорошие запросы:
- "сделай X теплее / ярче / плавнее"
- "убери [эффект] совсем"
- "слишком агрессивно, приглуши"
- "хочу больше контраста между A и B"

## После изменения:
Покажи что именно поменял (файл, строка, было → стало).
